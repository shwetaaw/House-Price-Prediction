# House Price Prediction Using Linear Regression

This project focuses on predicting house prices using **Multiple Linear Regression** based on various features such as size, number of rooms, year built, and location.  
It is developed as an academic project for **M.Sc. Data Science**.

---

## Introduction

The real estate market is influenced by various factors, including location, house size, number of rooms, and other amenities. This project aims to analyze a dataset containing house prices and their attributes to develop a predictive model for estimating house prices using **linear regression**.

By analyzing house price data, we can:

- Understand market trends  
- Identify key factors affecting house prices  
- Make informed decisions for buyers, sellers, and investors  

The study involves regression analysis to establish relationships between various features and house prices.

---

## Variable Information

The main variables used in this project are:

- **Price** (Continuous) – Selling price of the house  
- **Size** (Continuous) – Total area in square feet  
- **Bedrooms** (Discrete) – Number of bedrooms  
- **Bathrooms** (Discrete) – Number of bathrooms  
- **Floors** (Discrete) – Number of floors  
- **Year Built** (Discrete) – Year the house was constructed  
- **Location** (Categorical) – Neighborhood / area of the house  

---

## Objectives

- **Identify Key Factors Affecting House Prices**  
  Understand how factors like size, location, and number of rooms impact house prices.

- **Build a Predictive Model**  
  Develop a multiple linear regression model to estimate house prices based on the dataset.

- **Evaluate Model Performance**  
  Assess the accuracy of the model using metrics like:
  - R-squared (R²)
  - Root Mean Squared Error (RMSE)
